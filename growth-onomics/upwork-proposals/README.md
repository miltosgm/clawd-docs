# Upwork Proposal System

## How to Use
1. Find a job matching our services
2. Pick the right template below
3. Customize the [BRACKETS] with job-specific details
4. Send

## Proposal Templates
- `seo-proposal.md` - For SEO/organic traffic jobs
- `aeo-proposal.md` - For AI search optimization jobs
- `content-marketing-proposal.md` - For content/blog jobs
- `linkedin-leadgen-proposal.md` - For lead generation jobs

## Tracking
Log all proposals in `tracker.md`
